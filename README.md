# The Harvest Cup

Website for the Lowell Herb Company's Harvest Cup — a cannabis industry golf and cornhole invitational at Red Hook Golf Club, October 26, 2026.

## Site

Single-page static site deployed on **GitHub Pages**.

### Sections
- Hero with countdown
- Overview — About the event
- How It Works — Lottery, play, cause
- About — The vision and Hudson Valley region
- Event — Format, schedule, what to expect
- Prizes — Cash payout breakdown
- Music — Live lineup
- Cornhole — Tournament format
- Venue — Red Hook Golf Club details
- Sponsors — Brand partner cards with modals
- Charity — Last Prisoner Project
- Exhibitors — Grower showcase call
- Lottery Registration — Multi-step form
- FAQ — Accordion + AI event bot

## Tech
Static HTML/CSS. Custom fonts (Howards Four/Eight/Nine + Gotham). GitHub Pages hosting.

## Fonts
- Howards Four (display headlines)
- Howards Eight (section headings)
- Howards Nine (script accent)
- Gotham (body, ui)
