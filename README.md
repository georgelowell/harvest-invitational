# The Lowell Harvest Invitational

Website for the Lowell Herb Company Harvest Invitational golf event at Red Hook Golf Club, August 31, 2026.

## Pages
- `/` — Home / Hero
- `/event.html` — Full event details and schedule
- `/venue.html` — Red Hook Golf Club info
- `/sponsors.html` — Partner directory
- `/sponsors/*.html` — Individual partner about pages
- `/lottery.html` — Instagram lottery entry info
- `/faq.html` — Frequently asked questions

Static HTML/CSS. GitHub Pages hosting. Custom domain ready (CNAME file included).
